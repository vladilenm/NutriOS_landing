# NutriOS — Landing Page

Продающий лендинг для **NutriOS** — персональной операционной системы питания для занятых
разработчиков и специалистов умственного труда. Он помогает превратить хаос питания в простую
систему: планировать блюда, помнить предпочтения, формировать списки покупок и улучшать
рекомендации через обратную связь.

Это статический демо-лендинг, собранный по [SPEC.md](./SPEC.md), чтобы показать, как из
структурированного брифа собрать современный продуктовый landing page. Весь текст — на русском языке.

## Стек

- **Vite 7** + **React 19**
- Чистый CSS (дизайн-система на CSS-переменных, без UI-фреймворков)
- [lucide-react](https://lucide.dev) — иконки
- Шрифты: **Fraunces** (display) + **Manrope** (текст)
- Появление секций — через `IntersectionObserver` (без тяжёлых анимационных библиотек)

## Запуск

```bash
npm install
npm run dev      # локальная разработка → http://localhost:5173
npm run build    # продакшн-сборка в dist/
npm run preview  # предпросмотр собранного билда
```

## Структура

7 экранов согласно спецификации:

1. Навигация + Hero — с плавающим app-mockup
2. Проблема — почему питание ощущается хаотичным (4 карточки)
3. Как это работает — 4 шага с progress-линией
4. Командный центр — статичный dashboard-mockup (чат, план, покупки, память)
5. Возможности — 6 feature-карточек
6. Персонализация — цикл «План → Еда → Реакция → Улучшение»
7. Безопасность + финальный CTA + Footer

```
src/
├── App.jsx                 # сборка секций
├── main.jsx                # точка входа
├── styles/index.css        # дизайн-система + все стили
└── components/
    ├── Reveal.jsx          # обёртка для scroll-reveal
    ├── Header.jsx
    ├── Hero.jsx
    ├── Problem.jsx
    ├── HowItWorks.jsx
    ├── CommandCenter.jsx
    ├── Features.jsx
    ├── Personalization.jsx
    ├── Safety.jsx          # блок безопасности + финальный CTA
    └── Footer.jsx
```

## Дизайн

Направление — **calm premium wellness OS**: мягкие светлые фоны, оливково-зелёные акценты,
много воздуха, аккуратные карточки, ощущение продуктового dashboard. Адаптивно для desktop,
tablet и mobile. Уважает `prefers-reduced-motion`.

> NutriOS — ассистент для благополучия и обучения. Он помогает с ежедневным планированием и
> поддержкой привычек, но не заменяет профессиональную медицинскую консультацию.
