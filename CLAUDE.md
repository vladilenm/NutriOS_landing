# CLAUDE.md

Это главный файл для агентов по текущему проекту.

## Что это

Статический продающий лендинг **NutriOS** (Vite 7 + React 19), собранный по [SPEC.md](./SPEC.md). Одна страница, без роутинга, без стейт-менеджмента, без бэкенда. Весь пользовательский текст — на русском языке; сохраняй язык и тон существующего контента при правках.

[SPEC.md](./SPEC.md) — исходный бриф (структура секций, копирайтинг, дизайн-направление). Это источник истины по контенту: при изменениях содержания сверяйся с ним.

## Команды

```bash
npm run dev      # dev-сервер → http://localhost:5173
npm run build    # прод-сборка в dist/
npm run preview  # предпросмотр собранного билда
```

Линтера и тестов в проекте нет.

## Архитектура

- [src/App.jsx](src/App.jsx) — линейно рендерит секции в фиксированном порядке (Header → Hero → Problem → HowItWorks → CommandCenter → Features → Personalization → Safety → Footer). Порядок секций = порядок на странице; чтобы переставить/добавить экран, правь этот файл.
- Каждая секция — самодостаточный компонент в [src/components/](src/components/). Данные (списки карточек, шаги, фичи) объявлены как локальные константы внутри своего компонента, не выносятся наружу.
- [src/components/Reveal.jsx](src/components/Reveal.jsx) — единственная нетривиальная абстракция. Обёртка для scroll-reveal через `IntersectionObserver`: добавляет класс `.in` при попадании в вьюпорт. Проп `delay` (мс) даёт каскадное появление внутри секции, `as` меняет тег. Оборачивай новые появляющиеся блоки в `Reveal` вместо ручных наблюдателей.
- Никаких CSS-модулей и UI-фреймворков: **все** стили в одном файле [src/styles/index.css](src/styles/index.css), импортируется один раз в [src/main.jsx](src/main.jsx).

## Стили

Дизайн-система живёт в `:root` в начале [src/styles/index.css](src/styles/index.css) как CSS-переменные (палитра, радиусы `--r-*`, тени `--shadow-*`, `--font-display`/`--font-sans`, `--container`, `--gutter`, `--ease`). Используй эти токены вместо хардкода цветов/размеров. Классы именуются в стиле BEM (`.mock-card__head`, `.mock-card--offset`).

Направление — «calm premium wellness OS»: мягкие светлые фоны, оливково-зелёные акценты. Адаптив (desktop/tablet/mobile) и `prefers-reduced-motion` уже учтены в CSS — при добавлении анимаций уважай reduced-motion.

Иконки — только из [lucide-react](https://lucide.dev). Шрифты (Fraunces + Manrope) подключены через Google Fonts в [index.html](index.html), не через npm.
